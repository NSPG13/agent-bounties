This Node.js CLI script selects the safest funded bounty from a canonical feed using only built-in modules (https, fs, path, url). It fetches a JSON feed of bounties, validates each bounty against deterministic criteria (minimum funding, open status, required fields), calculates a safety score based on factors like funding amount, verifier reward, claim bond, clear requirements, labels, deadline, and commit immutability, then outputs the safest bounty as JSON for agent consumption. The script is dependency-free and designed for autonomous agent use.
